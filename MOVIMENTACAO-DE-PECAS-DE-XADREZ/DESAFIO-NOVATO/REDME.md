♟️ Movimentação de Peças de Xadrez em C
    Desafio Nível Novato-

Este projeto foi desenvolvido como desafio acadêmico para praticar estruturas de repetição na linguagem C.

O objetivo é simular o movimento de três peças do xadrez utilizando diferentes tipos de loops: 
for, 
while,
do-while.

🧠 Lógica do Programa

O programa simula o movimento de três peças:

Torre

Rainha

Bispo

Cada peça utiliza uma estrutura de repetição diferente, conforme solicitado no desafio.

🔁 Estruturas Utilizadas
🔹 Torre — for

Simula 5 movimentos para a direita.

O loop for foi escolhido por ser ideal quando o número de repetições é conhecido previamente.

🔹 Rainha — do-while

Simula 8 movimentos para a esquerda.

O do-while garante que o bloco execute pelo menos uma vez antes da verificação da condição.

🔹 Bispo — while

Simula 5 movimentos na diagonal (cima e direita).

O while foi utilizado com controle manual do contador.


📌 Controle de Variáveis

Foi utilizada uma única variável contadora (i).

O contador é reinicializado antes de cada estrutura de repetição para evitar interferência entre os loops.

🖥️ Saída do Programa

O programa imprime no console:

O nome da peça

A direção do movimento

A numeração de cada passo

Exemplo:

=== Movimento da Torre ===
Torre para Direita 1x
Torre para Direita 2x
...
